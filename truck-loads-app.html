Truck Loads App

App Store Google Play

Features

  • Load Management: Track loads with full CRUD functionality and status workflow.
  • Expense Tracking: Log expenses and attach receipt photos.
  • Reporting & Invoicing: Generate professional PDF invoices and settlement reports.
  • Dashboard & KPIs: View key metrics like Actual Revenue and Net Profit.
  • Data Management: Import and export all app data with JSON files.
  • Security: Optional PIN lock screen to protect sensitive financial data.

Technology Stack

  • Framework: .NET MAUI
  • Language: C#
  • Architecture: Model-View-ViewModel (MVVM)
  • Database: Local SQLite database for offline functionality
